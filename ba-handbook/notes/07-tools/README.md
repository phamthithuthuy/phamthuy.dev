# 07 — Tools

> Nhóm note về công cụ và literacy kỹ thuật vừa đủ cho BA: quản lý backlog (Jira), tài liệu (Confluence), thiết kế giao diện (Figma/draw.io), và đọc/viết SQL cơ bản. Mục tiêu là dùng được công cụ phục vụ công việc BA, không phải trở thành chuyên gia kỹ thuật.

## Current scope

Chưa có note thực hành. Source literacy kỹ thuật (SQL, security awareness, role-permission matrix) hiện nằm ở `collections/supporting-itba/` và các module `library/14-security/`, `library/16-sql/`, `library/12-role-permission-matrix/`.

## Planned scope

- Jira & Confluence cho BA — quản lý backlog và viết tài liệu.
- Figma / draw.io cho BA — wireframe và diagram.
- SQL cho BA — truy vấn dữ liệu để verify và phân tích.
- Security awareness — đủ để viết test case cơ bản (SQL injection, XSS) và ma trận phân quyền.
